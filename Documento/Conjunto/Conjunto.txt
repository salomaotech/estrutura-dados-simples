Conjunto ou Set

Tipo de estrutura de dados onde podemos adicionar dados de forma sequencial, porém teremos a garantia de que não haverá dados duplicados.
Os dados podem ou não estar na ordem em que foram inseridos dependendo da implementação.


Métodos genéricos

Add - Adiciona itens ao conjunto
Remove - Remove um elemento do conjunto
Contains - Checa se um elemento já está no conjunto
Size - Retorna o tamanho do conjunto
Elementos - Retorna os elementos do conjunto
